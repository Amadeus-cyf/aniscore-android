## Aniscore
[Aniscore](https://aniscore.herokuapp.com)是一个番剧评分平台。仿照bilibili UI设计。Web版于2019年7月开发完成，项目仍在维护。

## AniscoreAndroid
安卓手机app初始版。后续会加入更多功能，当前版本为1.0.0。
